## Installing

### Requirements
- Install Node.js (https://nodejs.org)
- Install Gulp (https://gulpjs.com/docs/en/getting-started/quick-start)

### Environment
```
$ git clone https://github.com/robertsaupe/node-webdev-basic.git
$ cd node-webdev-basic/
$ npm i
```

## Getting started
live develop:
```
$ npm run dev_build
```

just build project:
```
$ npm run build
```

clear builds:
```
$ npm run clear
```
